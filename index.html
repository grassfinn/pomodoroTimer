What is a Pomodoro Timer?

A Pomodoro Timer is a time management technique developed by Francesco Cirillo. This method is to encourages one to work on a task for the timer's duration and when the timer is complete, the user takes a short break. Franceso Cirillo orignally used a tomato kitchen timer when using this method. The word pomodoro comes from Itlian word tomato

I made this app because I needed a way to keep myself accountable for the time I procrastinate while I should be learning, and I want to create my own tools. I hope this site can be as helpful to others as it has been for me. I plan to add a todo list feature as well!
